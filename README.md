Minimum requirements;

```
Java 8, Git
```

Clone repo;

```
git clone git@github.com:johnlayton/dawsonroad.git
```

Change to project directory;

```
cd dawsonroad
```


Build shadow jar;

```
./gradlew shadowJar

```


Run application;

```
java -jar ./build/libs/dawsonroad-all.jar <players> <count>
```
